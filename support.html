Frequently Asked Questions

How do I get an activation code?
  1. Fill out our Google Form
  2. Wait for your activation code via email
  3. Enter the code in the extension
The extension isn't working on my platform
Make sure you're on a supported platform:
  • GMGN.ai (SOL & BNB chains)
  • Axiom.trade (SOL & BNB chains)
Try refreshing the page and ensure your activation code is valid.
My trading data disappeared
All data is stored locally in your browser. Data loss can occur if:
  • Browser cache is cleared
  • Extension is uninstalled/reinstalled
  • Browser data is reset
We recommend exporting your data regularly using the export feature.
Can I use this on multiple devices?
Each activation code is tied to a specific device. If you need to use the extension on multiple devices, please contact support for additional codes.
Is my data safe?
Yes! All your trading data is stored locally in your browser. We don't collect or store any of your trading information on our servers. Only activation codes are verified online.

Installation Issues

Extension won't load
  1. Make sure you're using Chrome browser
  2. Enable "Developer mode" in chrome://extensions/
  3. Try reloading the extension
  4. Check if your activation code is entered correctly

Performance Issues

Prices not updating
  • Check your internet connection
  • Refresh the page
  • Make sure you're on a supported token page
  • Try disabling other extensions temporarily
